# Admin Backend

This is the admin backend for RareFinds.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Configure environment variables:
    Copy `.env.example` to `.env` and update the values.
    (Already done if you used the scaffold tool)

## Running the Server

Start the development server:
```bash
npm run dev
```

## Testing

Run the login test script:
```bash
npm run test-login
```
